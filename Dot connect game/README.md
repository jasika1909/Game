# Dot Connect Game 🎮

## Project Overview

Dot Connect Game is an interactive puzzle game developed using Python where players connect matching colored dots without crossing paths. The objective is to fill the entire grid by creating valid connections between dot pairs while maintaining a clear and logical path structure.

This project demonstrates problem-solving logic, game development fundamentals, event handling, and graphical user interface implementation.
